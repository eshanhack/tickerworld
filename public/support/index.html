Support

Let’s get you back into the world.

Check the service status first. Tickerworld is designed to fall back to a truthful solo experience when multiplayer or news is unavailable.

Quick fixes

  • Refresh once, then try the direct market link again.
  • Tap the entry button before expecting sound; browsers require a gesture.
  • If performance is low, choose the low quality tier in settings.
  • If WebGL was lost, use the retry card instead of repeatedly refreshing.
  • Clear only Tickerworld site data if local preferences or blocks appear corrupted.

Contact

For private privacy questions, use privacy@tickerworld.io. For reproducible bugs that contain no personal information, use the public issue tracker. Never post party tokens, wallet information, chat evidence, IP addresses, or personal details in a public issue.
